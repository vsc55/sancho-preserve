# Sancho Preserve

This repository is an archival preservation of the Sancho GUI client for MLDonkey.

## Status

- Original software: abandoned
- Version: 0.9.4-23 (source snapshot)
- Language: Java (SWT)
- Runtime: legacy JVM required

## Purpose

This project exists only for preservation and reproducibility purposes.
It is not actively maintained.

## History

Sancho was originally developed in the early 2000s as a graphical interface for the MLDonkey peer-to-peer client.

This repository contains a frozen snapshot of the original source tree.
